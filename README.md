# Movie Database
https://movie-database.jamespanter.co.uk/

## About
Tech test at Nology. The main aim consisted of fetching data from an authenticated API. Extended with the implementation of lazy loading.

## Tech stack
* React

## How to use
1. Clone and download repo.
2. Run `yarn install`
3. Run `yarn start`
